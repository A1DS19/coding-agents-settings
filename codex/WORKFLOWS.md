# Codex Workflows

## Bug Fix Workflow
1. Reproduce and isolate root cause.
2. Add/adjust failing test when feasible.
3. Implement smallest safe fix.
4. Run targeted + relevant regression tests.
5. Summarize impact and edge cases.

## Feature Workflow
1. Confirm acceptance criteria.
2. Implement in small commits/steps.
3. Add tests for critical paths.
4. Validate backward compatibility.
5. Document usage and operational notes.

## Review Workflow
1. Prioritize correctness and regressions.
2. Check security/performance implications.
3. Confirm tests actually cover behavior.
4. Leave actionable findings with file references.
